Spevents

Real-time 3D photo gallery for special events. This project creates immersive photo experiences using modern web technologies.

Technologies Used

  • TypeScript for type-safe development
  • React for the user interface
  • AWS S3 for cloud storage
  • Three.js for 3D rendering

Features

  • Real-time photo uploads
  • 3D gallery visualization
  • Responsive design
  • Cloud-based storage
